# Product Overview

Mindful Nest (सजग बसेरा / Raksha) is a mental health and emotional wellness web application designed for Indian youth, particularly students facing academic and social pressures.

## Core Purpose
A safe, private space for emotional reflection and support, rooted in Indian cultural values of resilience, family, and community.

## Key Features
- Raksha Chat: 5-layer emotional processing system (emotion → situation → thoughts → regulation → reflection)
- Journal: Personal reflection and emotional pattern tracking
- Mood Tracker: Visual mood tracking with charts
- Community: Anonymous peer support and sharing
- Resources: Mental health information and crisis helplines
- Shanti Button: Quick access to breathing and grounding exercises

## Cultural Context
- Bilingual (English/Hindi) interface
- Indian aesthetic (mandala patterns, rangoli borders, warm earth tones)
- References to Indian philosophy (Metta Sutta, Om Shanti)
- Crisis resources specific to India (Kiran, iCall, AASRA helplines)

## Privacy & Safety
- All data stored locally (localStorage)
- No server communication
- Emergency resources prominently featured
- Clear disclaimer: not a replacement for professional therapy
